Adam optimizer initialization m1 = 0 (Initial 1st moment of gradient) m2 = 0 (Initial 2nd moment of gradient) n_iter = 0 update rule n_iter += 1 learning_rate *= sqrt(1 - beta2^n) / (1 - beta1^n) m1 = beta1 * m1 + (1 - beta1) * gradient m2 = beta2 * m2 + (1 - beta2) * gradient^2 param += learning_rate * m1 / (sqrt(m2) + epsilon)
